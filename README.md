## 🌿 The "Organic Green" Aesthetic

This portfolio features a custom-curated **Designer Light Theme**. It's not just a "light mode"—it's a sophisticated palette designed to feel natural, premium, and calm.

- **Neutral**: `Sand` (An earthy, warm-toned base)
- **Brand**: `Moss` (A deep, professional nature-inspired green)
- **Accent**: `Emerald` (Vibrant highlights for interaction)
- **Visuals**: `Translucent` surfaces with glassmorphism effects.

---

## 🚀 Key Features

- 💎 **Agent-First Design**: Built with Once UI's ultra-clean token system.
- 🌑 **Dynamic Themes**: Seamless switching between light and dark modes.
- ✨ **Magic Effects**: Cursor-tracking masks, interactive gradients, and subtle noise textures.
- 📱 **Fully Responsive**: Flawless experience across all devices.
- ⚡ **Optimized Performance**: Built on Next.js 15 for lightning-fast delivery.
- 💌 **Mailchimp Integrated**: Ready-to-go newsletter subscription block.

## 🏗️ Getting Started

1. **Clone & Install**
   ```bash
   git clone https://github.com/saqlainartaz/portfolio.git
   npm install
   ```

2. **Launch Dev Server**
   ```bash
   npm run dev
   ```

3. **Customize**
   Head over to `src/app/resources/config.js` to tweak your magic.
